== Description ==

A plugin that lets you quickly and easily create a view of test result of health insurances.

Install, activate and then go to Krankenkassentest. You can select a category to view and select a slug of virtual page
to view on your site. Also you can include the shortcode [krankenkassentest] and place it on the site you want.

You can select the style to display the result. Select Bootstrap 3, Bootstrap 4 or own styles.

This plugin call a json-file from https://cdn.krankenkasseninfo.de to display the results.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `krankenkassentest` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Krankenkassentest in menu
